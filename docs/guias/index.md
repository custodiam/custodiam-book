# Guías técnicas

Esta sección agrupa las guías técnicas paso a paso para operar el sistema. Las guías internas del equipo se mantienen en el repositorio privado de documentación y se publican aquí de forma curada conforme alcanzan versión final.

## Guías incluidas

!!! info "Sección en construcción"
    Las guías están siendo migradas de forma controlada desde el repo interno. La primera oleada de publicación incluirá las guías más demandadas por contributors externos:

    - **Guía 00 — Setup de repositorios** (cómo crear el polyrepo desde cero).
    - **Guía 21 — Setup FastAPI con uv** (backend en local).
    - **Guía 03 — Docker Compose local** (stack completo).
    - **Guía 10 — Configuración Keycloak** (realm + clientes OIDC).
    - **Guía 25 — Flutter OIDC** (cliente OAuth + PKCE en web y móvil).

Mientras tanto, los recorridos rápidos de [Empezar](../empezar/index.md) cubren los pasos esenciales para arrancar cada componente.

## Estructura prevista de las guías

Cada guía publicada seguirá el mismo patrón estructural:

1. **Prerrequisitos** — qué tener instalado/configurado antes de empezar.
2. **Pasos numerados** — cada paso con su comando, screenshot si aplica, y verificación.
3. **Variables y configuración** — qué archivos editar y con qué valores.
4. **Troubleshooting** — errores comunes y cómo resolverlos.
5. **Próximos pasos** — qué guía leer después.
